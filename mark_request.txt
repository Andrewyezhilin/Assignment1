## Design Excellence

If you have completed a "design excellence" task, please write what
the task was below (note, it MUST be on the list in the assignment or
approved by COMP6991 staff).

> Achieved over 80% application test coverage using cargo-tarpaulin.

Please write a paragraph or two (max. 250 words) describing how your design was
influenced by this challenge, with specific reference to sections of your code.

> I structured the program as a library and a small command-line application to
> make the scoring logic independently testable. `src/main.rs` is responsible for
> command-line parsing, YAML input, and displaying the final result, while
> `src/lib.rs` exposes the `score` function and contains the poker-hand and Joker
> scoring implementation.
> I added 25 end-to-end tests in `tests/cli.rs`. These tests exercise all
> poker-hand categories, card enhancements and editions, Joker activation phases,
> Blueprint chains, retriggers, Four Fingers, Shortcut, Smeared Joker, Flower Pot,
> file input, and standard input. The tests use the public `score` interface rather
> than exposing private implementation details solely for testing. cargo-tarpaulin
> reports 86.32% line coverage, with 385 of 446 measured lines covered.
